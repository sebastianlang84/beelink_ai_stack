# Transcript Analysis Report (Stocks)

## Report Metadata
- **Run ID:** …
- **Run Date:** …
- **Report Generated:** …
- **Topic:** …

## 1. Executive Summary
- 3–5 sentences: scope (videos/creators), time window/recency, main themes.
- Avoid “fundamental analysis” voice; explicitly frame as channel-content digest.

## 2. Key Findings
- 5–12 bullets, grouped by theme.
- For hard claims include source (channel + video title + video id) and data quality (ok/low/truncated).

## 3. Market Sentiment
- For each top stock: **Bull case**, **Bear case**, **Catalysts**, **Invalidation criteria** (1–3 bullets each).
- Make signal strength visible: mentions/videos/creators + confidence (high/medium/low, justified by coverage + quality).

## 4. Top Stocks
- Uniform format per line: `Company (TICKER) — mentions: X — videos: Y — channels: Z — recent30 videos: R — stale_days: S`
- No duplicates (e.g. TSLA only once).

## 5. Channel Coverage
- Table/ranking: channel → transcripts/summaries/videos-with-mentions/mentions.
- Call out channels with transcripts but missing summaries (if any).

## 6. Mini Factsheets (channel-content only)
- For each top stock, list 3–8 bullets with numbers/multiples **only if present in the material**; otherwise omit/mark unknown.
- Do not invent or look up external market data (market cap, peers, etc.).

## 7. Stocks Mentioned (non Deep-Dive)
- Short summary per stock (1–3 sentences) + source (channel + video title + video id).

## 8. Other Insights
- Important insights outside macro/crypto & stocks (1–2 sentences) + source (channel + video title + video id).
